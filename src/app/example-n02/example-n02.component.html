Size And Position Of The Preview

You can set any size of preview image via CSS/SCSS, red-zoom automatically detect it, and you do not need to edit ts/html files for this purpose. This also means that you can set different sizes of preview depend on screen size using CSS media queries.

Use the following CSS variables to position the preview relative to the thumbnail:

  • --red-zoom-thumb-[x/y] – Thumbnail position relative to the page.
  • --red-zoom-thumb-[w/h] – Thumbnail size in pixels.
